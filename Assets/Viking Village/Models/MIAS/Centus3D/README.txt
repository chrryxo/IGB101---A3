Thank you for purchasing this asset pack!

This package includes:

- 1 FBX model
- 4K and 2K textures in the following formats:
  - Generic PBR (BaseColor, Normal, Roughness, AO, Metallic, Height)
  - Unity URP
  - Unity HDRP (Mask Map included)
  - Unreal Engine packed textures

Instructions:
- Import the FBX file into your engine of choice.
- Apply the appropriate texture set depending on your pipeline.
- In Unity HDRP, use the Mask Map and assign it to the "Mask Map" slot under the HDRP Lit Shader.
- In Unreal Engine, use the packed textures (R: AO, G: Roughness, B: Metallic) for best performance.
